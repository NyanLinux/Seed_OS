## Why in rust? 

1. Rust is one of the most energy efficient languages
2. It's a great way to learn
3. Why not?

## Installation

OS X & Linux:

```sh
cargo build --target thumbv7em-none-eabihf 
```

Windows:

```sh
cargo rustc -- -C link-args="/ENTRY:_start /SUBSYSTEM:console"
```
